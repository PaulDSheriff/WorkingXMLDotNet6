To run this project in Visual Studio Code, open the XMLSamples.code-workspace
  When prompted to add "Required Assets", Click the 'Yes' button
  Run the project
  Check the DEBUG CONSOLE window for the output
To run this project in Visual Studio 2022, open the XMLSamples.sln
  Run the project
  A console window shows the output


CreateViewModel Samples
--------------------------------------------------
CreateEmptyDocument() - Write code to build a new XML document that is empty
CreateProductDocument() - Write code to build a new XML document with a new product element
CreateProductDocumentWithAttribute() - Write code to build a new XML document with a new product element and attribute
CreateNestedXmlDocument() - Write code to create a nested XML document
ParseStringIntoXDocument() - Create an XML string and parse that into an XML document using XDocument
ParseStringIntoXElement() - Create an XML string and parse that into an XML document using XElement


AddEditDeleteViewModel Samples
--------------------------------------------------
AddNewNode() - Write code to create a new XElement object and add it to an existing XML document
UpdateNode() - Write code to retrieve a single node from a document, edit some elements, then display the changed elements
DeleteNode() - Write code to locate a specific node, then delete that node from the XML document
